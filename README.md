# Weltspiegel Template

Joomla 5/6 site template for Weltspiegel Cottbus cinema.

## Installation

1. Zip the template directory
2. Install via Joomla Administrator: System → Templates → Install
3. Set as default template: System → Site Templates → Weltspiegel → Make Default

## Development

This is a minimal starting point. Add your custom styles, scripts, and structure as needed.
